# **Sets**
- a set of unique elements, sort not needed.
- cannot insert more than there is in the universal set.

## **Implementations**
  - Arrays
  - Linked Lists
  - Cursor-Based
  - Bit Vector
  - Computer Word
 
## **Operations**
  - Insert
  - Delete
  - Find / Comparison
  - Union (OR)
  - Intersection (AND) 
  - Difference (XOR)
  - Subset
